ColorHunter

ColorHunter is a plugin to get dominant color and make a color palette from a image.
In order to use ColorHunter, you must first load the plugin at the top of your script using the LoadPlugin method like this:

 app.LoadPlugin( "ColorHunter" );

For getting dominant color

 var Color = getDominantColor( img );

Where:

img - image src


For getting colors palette

 var Color = getPalette( img, opacity, count );

Where:

img - image src
opacity - count of opacity added to the color
count - how many colors wanted, you can set 0 to 1, where 1 = 100, 0.1 = 10
